Spacing & Typography System

A universal system for consistent spacing and type hierarchy across any project.

This page defines the core spacing and typography rules that apply to every screen, section, and component. Use the sidebar to jump to any topic.

Wireframes vs UI

This is an important distinction — wireframes get us aligned on structure, but they don't define the final details.

Wireframes Define
  • Structure
  • Hierarchy
  • Intent
Wireframes Do Not Define
  • Exact spacing
  • Exact font sizes
  • Final visual consistency

That's why we need this system — it fills the gap between what the wireframes show and what actually gets built. Instead of guessing spacing or font sizes from the wireframe, you use the rules below. The goal is to avoid mixing sizes across sections.

For typography specifically, we're standardizing on a fixed scale (you'll see it in the next section). This means every heading, body text, and small label uses a specific size from the scale — no in-between values.

Quick Implementation Rules

Tight groups, generous section breaks. These four values cover 95% of every layout.

16px
Text Relationships

Title to subtitle, heading to description, label to value.

32px
Text to Content

Description to cards, heading to grid, text to visual content.

48px
Inner Sections

Between content groups within a section. Moderate breathing room.

80px
Section Spacing

Between major page sections. Creates the premium, editorial feel.

Exception

Hero Sections Don't Follow These Rules

Hero sections are the one place where you can break the global spacing and typography system. They're the first thing a user sees — they need to set the tone, communicate the brand, and create impact. That means hero sections can use:

  • Larger or custom font sizes outside the type scale (e.g., display type at 56-80px)
  • Tighter or looser letter-spacing for visual impact
  • Custom spacing that gives the section more breathing room
  • Brand-specific styling, colors, or layout choices

Everything else on the page — service sections, cards, blog layouts, footers — follows the system. But the hero is where branding lives, so treat it as its own thing.

When in doubt: if it's the hero, match the brand guidelines. If it's anything else, use the spacing and type scale above.

That's It

Once a team aligns on these rules, they can be applied consistently across every page and component without needing detailed QA on every section. Standardize everything using four values — 16px, 32px, 48px, and 80px — and move fast. The generous section breaks are what give any site that premium, editorial feel.
